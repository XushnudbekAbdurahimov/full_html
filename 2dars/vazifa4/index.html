Bugungi mavzu:       HTML da shriftlar

                                                                Dastlabki tushuncha

           Ko'p so'zlar aytilishi ham yozilishi ham biroz qiyin bo'ladi.Ammo HTML dasturlash tilida bunga chora topilgan!Xo'sh chorasi bormi degan bo'lsangiz,men uni sizga aytib o'taman.HTML dasturlash tilida ba'zi bir so'zlar uzun bo'lganligi uchun shu so'zni bosh harfi yoki ikkita harf birga olinadi.Bunga misol "Superscript" so'zi.Aytilishi ham yozilishi ham biroz qiyinroq.Shuning uchun bizga oson bo'lishi uchun oddiygina "sup" deb atalgan.

                                                                               Hozir men sizga ularni sanab o'taman.

                   1.Italic so'zi.Bu so'zning qisqartmasi "i".Bu element so'zlarni yotiq xolatda qilib beradi

2.Bold so'zi.Bu so'zning qisqartmasi "b".Bu element so'zlarni qalinlashtirib beradi

                   3.Emphasize so'zi.Bu so'zning qisqartmasi "em".Bu element so'zlarni yotiq qilib beradi

4.Important so'zi.Bu so'zning qisqartmasi "strong".Bu element so'zlarni qalinlashtirib beradi

                   5.Small so'zi.Bu so'zning qisqartmasi "small".Bu element so'zlarni kichik qilib beradi

6.Delete so'zi.Bu so'zning qisqartmasi "del".Bu element so'zlarni ustidan chiziq tortadi

                   7.Inserted so'zi.Bu so'zning qisqartmasi "ins".Bu element so'zlarni tagiga chiziq tortadi

8.Superscript so'zi.Bu so'zning qisqartmasi "sup".Bu element so'zlarni tepasidan soz yozish uchun ishlatiladi

                   9.Subscript so'zi.Bu so'zning qisqartmasi "sub".Bu element so'zlarni tagidan soz yozish uchun ishlatiladi

10.Underline so'zi.Bu so'zning qisqartmasi "u".Bu element so'zlarni tagiga chiziq tortadi

                   11.Marked text so'zi.Bu so'zning qisqartmasi "mark".Bu element so'zlarni belgilash uchun ishlatiladi
